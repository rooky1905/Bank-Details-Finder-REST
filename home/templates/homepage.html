Hey Guys In order to go ahead and use this REST-API Service you need to do the following...


    1. Add data CSV file

    2. After adding the data, head over to http://127.0.0.1:8000/api/ifsc/[ifsc_code] and bank details will be displayed in json format

    3. Then to see all banks in a city with the name; head over to http://127.0.0.1:8000/api/branches/[city]/[bank-name] and a list will be displayed in json format


    [] represents placeholder where the specified data needs to be put in by user
